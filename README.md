#  OutfitGram

OutfitGram is a modern and responsive fashion web application built with **Next.js**. It allows users to browse outfit collections through a clean, attractive, and user-friendly interface.

---

##  Features

- Responsive Design
- Modern User Interface
- Category-based Outfit Browsing
- Search Bar
- Reusable Components
- Fast Navigation using Next.js
- Clean Folder Structure

---
##  Technologies Used

- Next.js
- React
- TypeScript
- CSS
- HTML

##  Installation

Clone the repository:

```bash
git clone https://github.com/TanzilaAli/OutfitGram.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
